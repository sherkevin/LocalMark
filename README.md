## Usage

### First Launch

1. Launch uTools and search for `md`, `Markdown`, or `ç¬”è®°`
2. Select "LocalMark" from the results
3. On first launch, you'll be prompted to select a root folder for your notes
4. Choose or create a folder where you want to store your Markdown files
5. Start writing!

### Basic Operations

#### Create Files/Folders
- **From Sidebar**: Click the file/folder icons in the EXPLORER header
- **Right-click Menu**: Right-click in the file list to create new items
- **Keyboard**: Use uTools shortcuts for quick access

#### Edit Files
- Click any `.md` file in the sidebar to open it
- Start typing - changes are auto-saved
- Use the toolbar for formatting (bold, italic, lists, etc.)

#### Search
- Type in the search box at the top of the sidebar
- Search matches both file names and content
- Click any result to open the file

#### File Management

#### Organize Files
- Drag files to move them between folders

## Migration

### Export Your Notes
Simply copy your notes folder to any location. All files are standard Markdown, compatible with:
- Obsidian
- Typora
- VS Code
- Any Markdown editor

### Import Existing Notes
1. Select your existing notes folder as the root folder
2. All `.md` files will be immediately available
3. No conversion needed!

## Keyboard Shortcuts

- `Ctrl/Cmd + S`: Save (auto-save enabled)
- `Ctrl/Cmd + Z`: Undo
- `Ctrl/Cmd + Y`: Redo
- `Ctrl/Cmd + F`: Search (in uTools)
- `Esc`: Close plugin

## Technical Details

- **Editor**: Vditor (WYSIWYG mode)
- **Storage**: Local file system
- **Format**: Standard Markdown (.md)
- **Platform**: Windows, macOS, Linux (via uTools)

## Requirements

- uTools (latest version recommended)
- No additional dependencies
- No internet connection required

## Privacy

- **100% Local**: All data stays on your device
- **No Telemetry**: No data collection or tracking
- **No Account**: No registration required

## Troubleshooting

### Plugin won't start
- Ensure uTools is up to date
- Check that all files in the plugin folder are intact
- Try reinstalling the plugin

### Files not saving
- Check folder permissions
- Ensure the root folder is accessible
- Verify disk space availability

### Search not working
- Make sure you're searching in a valid root folder
- Check that files have `.md` extension
- Try refreshing the file list
